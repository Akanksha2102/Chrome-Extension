# Chrome New Tab Dashboard Extension

A Manifest V3 Chrome extension that replaces the default new-tab page 
with a customizable dashboard.

## Features
- Custom wallpaper upload with file validation, or randomized rotation
- Add/remove custom website shortcuts with auto-fetched favicons
- Live clock and date display
- In-browser search using Chrome's native search API

## Tech Stack
JavaScript, jQuery, HTML5, CSS3, Chrome Extension APIs (Manifest V3)

## Installation
1. Clone this repo
2. Go to `chrome://extensions`
3. Enable Developer Mode
4. Click "Load unpacked" and select this folder
